HTML Text Formatting Tags

Heading Tags

<h1> to <h6> : Define HTML headings. <h1> defines the most important heading and <h6> defines the least important heading.

Bold and Strong Tags

<b> : Defines bold text.

<strong> : Defines text with strong importance.

Italic and Emphasis Tags

<i> : Defines italic text.

<em> : Defines emphasized text.

Preformatted Text Tag

<pre> : Defines preformatted text.

Marked Text Tag

<mark> : Defines marked or highlighted text.

Deleted and Inserted Text Tags

<del> : Defines deleted text.

<ins> : Defines inserted text.

Underlined Text Tag

<u> : Defines underlined text.

Subscript and Superscript Tags

<sub> : Defines subscript text.

<sup> : Defines superscript text.

Defining Term Tag

<dfn>: Defines a term that is being defined.

Paragraph, Line Break, and Horizontal Rule Tags

<p>: Defines a paragraph.

<br>: Inserts a single line break.

<hr>: Defines a thematic break between paragraph-level elements.
